# Flipper-Lite

DIY ESP32-S3 based portable multi-protocol security tool.

## Core Features
- Wi-Fi scanning and handshake capture
- BLE scanning
- USB HID payloads
- IR learn and replay
- NFC (PN532)
- Sub-GHz (CC1101)

## Hardware
- ESP32-S3 N16R8
- OLED display
- MicroSD storage
- Li-Po battery
- PN532 NFC
- CC1101 RF module
